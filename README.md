# 📚 学习习惯养成系统

帮助你培养学习习惯的全栈管理工具。

## ✨ 功能

| 模块 | 说明 |
|------|------|
| 📋 任务管理 | 创建/编辑/删除任务，支持优先级、主题、截止时间 |
| ✅ 每日打卡 | 完成任务打卡，连续打卡解锁龙印章 🐉 |
| 📈 数据分析 | 完成率、连续天数、日历热力图 |
| 🔔 提醒中心 | 截止提醒 + 科学复习提醒（第1/2/4/7天） |
| 🍅 番茄钟 | 15/25/45/60 分钟专注计时 |
| 👥 好友 & 聊天 | 添加好友，一对一实时聊天 |
| ⭐ 收藏 & 推荐 | 任务模板收藏 + 系统推荐任务库 |
| 💬 每日名言 | 首页随机名言，换一句激励自己 |

## 🚀 使用

```bash
# 双击根目录脚本即可
编译并启动.bat    # 编译 + 启动服务器
启动服务器.bat     # 仅启动（已编译过）
```

浏览器打开 `http://localhost:8080`
