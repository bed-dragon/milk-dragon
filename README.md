# 📚 学习习惯养成系统

帮助你培养学习习惯的全栈学习管理工具。C++ 后端 + 原生前端，单文件部署，开箱即用。

## ✨ 功能

| 模块 | 说明 |
|------|------|
| 📋 任务管理 | 创建/编辑/删除任务，支持优先级、主题、截止时间 |
| ✅ 每日打卡 | 任务打卡 + 每日签到，打卡日历热力图 |
| 📈 数据分析 | 完成率、连续打卡天数、日/周/月统计 |
| 🔔 提醒中心 | 截止提醒 + 科学复习提醒（第1/2/4/7天） |
| 🍅 番茄钟 | 15/25/45/60 分钟自由切换，跨页面持久化计时 |
| 👥 好友 & 聊天 | 添加好友，一对一实时聊天 |
| ⭐ 收藏 & 推荐 | 任务模板收藏 + 一键复用 |
| 👑 管理后台 | 用户管理、推荐任务库、系统统计 |
| 💬 每日名言 | 首页随机名言 |

## 🛠 技术栈

| 层 | 技术 |
|---|------|
| 后端 | C++17 + httplib + SQLite3 + nlohmann/json |
| 前端 | 原生 HTML/CSS/JS |
| 部署 | 单端口 8080 + ngrok 公网穿透 |

## 🚀 快速开始

### 环境要求

- Windows 系统
- MinGW-w64（g++ 支持 C++17）

### 编译 & 启动

```bash
# 双击根目录脚本
编译并启动.bat     # 编译 + 启动
启动服务器.bat      # 仅启动（已编译过）
联机启动.bat        # 启动 + ngrok 公网
```

或手动编译：

```bash
cd backend
gcc -c libs/sqlite3.c -o sqlite3.o
g++ -std=c++17 -Isrc -Ilibs src/*.cpp sqlite3.o -lws2_32 -o server.exe
server.exe
```

浏览器打开 `http://localhost:8080`

### 局域网联机

```
ipconfig                    # 查看本机 IP（如 192.168.1.105）
其他人访问 → http://你的IP:8080
```

### 公网联机

```bash
联机启动.bat                 # 或手动 ngrok http 8080
# 访问 ngrok 生成的公网地址
```

```
│   └── pages/            # 功能页面（13 个）
├── 编译并启动.bat
├── 联机启动.bat
└── README.md
```
